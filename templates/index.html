Cara menggunakan

1. ESP32 harus sudah terhubung dengan internet dan terhubung dengan broker.

2. Klik Tombol CHECK_STATE untuk memeriksa keadaan nyala/mati lampu LED

3. Klik Tombol TURN_ON untuk menyalakan lampu LED

4. Klik Tombol TURN_OFF untuk mematikan lampu LED

5. Keadaan lampu LED akan diperiksa setiap kali tombol CHECK_STATE, TURN_ON, atau TURN_OFF ditekan

6. Tombol CHECK_STATE perlu ditekan setiap tombol TURN_ON atau TURN_OFF ditekan agar aplikasi dapat menampilkan keadaan lampu LED yang tepat
